## Buy More, Save More Quantity Bundles in Shopify (No App)

Developed a custom “Buy More, Save More” quantity bundle system in Shopify without relying on any third-party apps. This feature encourages customers to purchase higher quantities by offering tiered discounts directly on the product page.

Customers can clearly see value-driven options like:

- Buy 2 → Save 10%  
- Buy 3 → Save 20% (Best Value)

Making it easier for them to choose higher bundles and increasing Average Order Value (AOV).

The solution was implemented using Liquid and JavaScript, ensuring it works seamlessly across Shopify themes while keeping performance fast and lightweight compared to app-based alternatives.

The UI is fully customizable, allowing merchants to highlight best offers, display dynamic pricing, and create a conversion-focused shopping experience without recurring app costs.

## Add FREE GIFT to Shopify Cart Without Any App

Built a custom Shopify feature that automatically adds a free gift to the cart when customers reach a specific spending threshold without using any third-party apps.

The solution includes a dynamic cart progress system that motivates users to unlock rewards like free shipping and free gifts.

Example:

- Free Shipping → $50  
- Free Gift → $70  

As customers add items, the progress updates in real time, encouraging higher spending and improving AOV.

Implemented using Liquid and JavaScript in the Shopify Dawn theme, this lightweight approach avoids app bloat, improves performance, and gives full control over customization.

## Product Engraving Personalization (No App)

Implemented a custom laser engraving personalization feature on the Shopify product page without using any third-party apps.

Customers can enable engraving, enter custom text, and the additional cost is automatically applied.

To overcome Shopify’s pricing limitations, engraving is added as a separate linked line item. The text remains visible in cart, checkout, and order.
